3）更改主题 Theme

在 Terminal 中输入命令Get-PoshThemes可查看所有可用的主题，在版本2中使用的命令为 Get-Theme。

❤找到喜欢的主题后，可以在之前的"脚本文件"[^1]中将主题名称替换成你喜欢主题的名称。

4）安装 Nerd Fonts 字体并应用

部分 Oh my posh 主题有一些特殊的字符，例如表示系统类型的徽标、GitHub 标志，这些字符需要特殊的字体支持。

这里我们需要安装 Meslo LGM NF 字体，这是 Oh my Posh 官方推荐使用的字体，使用微软推荐的Cascadia Code PL 和 Cascadia Mono PL 会出现图标不显示的问题。

Nerd fonts 字体官网：https://www.nerdfonts.com/

Nerd fonts 字体Github：https://github.com/ryanoasis/nerd-fonts

4 字体 Font

前面提到过，使用的字体是 Nerd Fonts 字体。但是，该字体仅支持西文字体，并不支持中文字体。

所以个人打算自行更改字体。即西文字符使用 Hasklig，中文字符使用霞鹜文楷，符号使用Nerd Fonts 字体。这里提供两种方案：

  • Hasklig：https://github.com/i-tu/Hasklig

    Hasklig 是大佬由 Source Code Pro 修改而来，支持部分连字。

  • 霞鹜文楷：https://github.com/lxgw/LxgwWenKai/

  • Nerd Fonts：https://www.nerdfonts.com/

  1. 使用软件 FontForge 完成。此方案较为繁琐，具体方法请自行百度。

    • 链接地址：https://fontforge.org/

  2. 快捷方案，使用 Warcraft Font Merger 完成。WFM 是一个字体合并/补全工具。操作简单，效果较为完美。

    • 链接地址：https://github.com/nowar-fonts/Warcraft-Font-Merger
